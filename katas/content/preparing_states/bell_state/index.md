**Input:** Two qubits in the $\ket{00}$ state (stored in an array of length 2).

**Goal:**  Change the state of the qubits to $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} \big (\ket{00} + \ket{11}\big)$.
